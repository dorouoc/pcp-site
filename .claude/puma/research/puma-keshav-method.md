# KESHAV THREE-PASS METHOD

- **Pass 1 (Overview)**: 
  - Read title, abstract, intro, headings, conclusion.
  - Identify **Category**, **Context**, and core **Contributions** of the paper.
- **Pass 2 (In-Depth)**:
  - Read entire paper (skip heavy math initially).
  - Note key ideas, methods, results, assumptions.
  - Mark sections for further review.
- **Pass 3 (Critical)**:
  - Re-implement core arguments, follow derivations, analyze experiments.
  - Answer the 5 C's: *Category, Context, Correctness, Contributions, Clarity*.
  - Create a **Permanent Note** summarizing the paper’s insights and relevance to PUMA.

## OUTPUT

- **Fleeting Note** (quick impressions from Pass 1).
- **Literature Note** (detailed summary and critique from Pass 2).
- **Permanent Notes** (synthesized knowledge from Pass 3).
