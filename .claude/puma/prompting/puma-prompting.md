# PROMPT ENGINEERING

- **Contextual Anchoring**: Provide clear role and background in prompts.
- **Chain-of-Thought (CoT)**: Ask the model to reason step-by-step for complex problems.
- **Zero-Shot vs Few-Shot**:
  - **Zero-Shot**: Direct instruction without examples when clear.
  - **Few-Shot**: Include examples to guide the response if needed.
- **RCOIF Framework**: Define Role, Context, Objective, Instructions, Format in prompts.
- **Iterative Refinement**: After an answer, ask the model to critique and improve (Self-correction).
- **One-Shot/Few-Shot**: Use sparingly when context is ambiguous.
- **Autodiagnosis & Improvement (AMI)**: Ask the agent to evaluate its own answer and refine.
- **Deconstruction & Reconstruction (DRCA)**: Break down tasks and rebuild solutions step-by-step.
- **Inverse Prompt Engineering (IIPR)**: Given an output, have the AI infer the prompt or logic.

## OUTPUT

- Document prompt versions and their results.
