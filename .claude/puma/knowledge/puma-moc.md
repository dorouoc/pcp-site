# MAP OF CONTENT (MOC)

- An MOC is an index/dashboard note linking related notes on a topic.
- Each major subject gets an MOC (e.g., "MOC: Multi-Agent PM Systems").
- MOC contents:
  - A clear title (e.g., "MOC: X topic").
  - A short description of the topic.
  - A list of links to relevant notes (Zettelkasten, project docs, resources).
- Update MOCs when new notes are added.

## PURPOSE

- Facilitate navigation of complex information.
- Ensure every note connects to a broader context.
- Act as entry points for research and review (start here for an overview).

## OUTPUT

- **MOC notes** in `80 - MOC`.
- Example: Add all prompt-engineering notes to "MOC-Prompts-Library.md".
