# RESEARCH TOOLS

Use the following platforms to find and organize relevant research:

- **ResearchRabbit**: Citation maps and topic exploration (app.researchrabbit.ai).
- **Semantic Scholar**: Scholarly search for AI, agents, project management.
- **Google Scholar**: Broad academic search.
- **Consensus**: AI-driven literature insights (consensus.app).
- **UnD3rmind**: Automated report summarization (undermind.ai).
- **Connected Papers**: Visualize research paper graphs.
- **LitMaps**: Track literature review progress.
- **Zotero**: Reference manager to collect and annotate papers.

## USAGE

- Formulate clear search queries (e.g., “LLM agents project management 2026”).
- Apply filters (years 2025–2026, fields like AI or CS).
- Collect PDFs and export citations to Zotero.
- Summarize key points in Literature Notes (30 - Resources).
- Tag references by topic/method (e.g., #LLM-Agent, #GTD).
