# ZOTERO INTEGRATION

- Use the **Zotero plugin** to sync references into Obsidian (or export BibTeX).
- Include Zotero keys in notes (e.g., `[@doe2026]` for citations).
- Organize references into Zotero collections (e.g., PUMA, Agents, PM).
- For each paper, create a note in `30-Resources` with:
  - Full citation (APA/MLA) from Zotero.
  - Summary of findings and relevance.
- Regularly update the Zotero library as new papers are found.

## OUTPUT

- **Bibliography**: Update `BIB-Master-APA7.md` in `62-Bibliographies`.
- **Literature Notes** linked to Zotero keys.
