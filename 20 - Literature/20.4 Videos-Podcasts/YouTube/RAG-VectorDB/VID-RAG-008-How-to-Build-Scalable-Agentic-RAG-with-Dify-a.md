## Summary

Official Dify tutorial demonstrating scalable agentic RAG using Qdrant as the vector store. Shows Qdrant's payload filtering capabilities for targeted retrieval (filter by metadata before semantic search), its multi-tenancy support, and its horizontal scaling. Includes performance benchmarks comparing filtered vs unfiltered retrieval.

---

## PUMA Relevance

The Qdrant payload filtering tutorial is directly applicable to PUMA Stage 4: filtering retrieved issues by Apache project (kafka, spark, etc.) before semantic search reduces noise in the triage classification context. The performance benchmarks inform PUMA's latency SLA estimates for Stage 4.
