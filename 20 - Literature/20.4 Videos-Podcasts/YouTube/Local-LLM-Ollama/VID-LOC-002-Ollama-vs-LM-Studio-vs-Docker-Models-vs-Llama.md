## Summary

Comprehensive comparison of four local inference platforms: Ollama (CLI + REST API), LM Studio (GUI + REST API), Docker Models (official Docker AI extension), and LlamaBarn (newer entrant). Evaluation criteria: API compatibility, model management, update mechanism, reproducibility (content-addressable models), multi-user support, and Docker integration.

---

## PUMA Relevance

Confirms PUMA's Ollama choice. Ollama wins on: content-addressable model registry (models are pinned by digest, guaranteeing identical outputs across machines — crucial for PUMA Constitution Article 1), Docker integration (Ollama image runs in PUMA's Docker Compose stack), and OpenAI API compatibility (LangGraph integrations work unchanged). LM Studio is confirmed as the valid fallback.
