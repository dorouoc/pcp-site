## Summary

Translation and commentary on Anthropic's internal guidance for building reliable agents. Key secrets: (1) start with the simplest possible agent and add complexity only when needed, (2) agents fail not from lack of capability but from ambiguous instructions, (3) tool design matters more than model choice, (4) HITL is not optional for production.

---

## PUMA Relevance

Anthropic's design principles are directly encoded in PUMA's Constitution (Article 4: HITL mandatory, Article 1: reproducibility over capability). The 'tool design > model choice' principle supports PUMA's decision to use Pydantic AI schema validation rather than chasing larger models. Critical for Stage 1–3 MVP design.
