## Summary

Andrej Karpathy demonstrates replacing traditional RAG pipelines with an Obsidian vault managed by Claude Code. The insight: rather than embedding documents into a vector store, a well-structured Obsidian graph provides richer context because LLMs can follow wikilinks, understand note hierarchies, and benefit from human-curated connections between concepts.
